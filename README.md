# 减脂计划 · 备餐 + 训练

一个**单文件、离线可用**的网页 App，帮你一次性规划一整周的减脂餐，并能搭配一套居家训练计划，饮食运动一起管，省去每天纠结"吃什么、练什么"的精神消耗。

## 功能

- **身体数据 → 热量方案**：输入身高、体重、体脂率、活动量，自动算出基础代谢(BMR)、每日总消耗(TDEE)、减脂目标热量与蛋白/脂肪/碳水分配。
- **饮食习惯 & 忌口**：不吃辣（内置菜谱全为不辣）、不吃牛肉/海鲜/内脏、自定义忌口（逗号分隔）、可选加餐。
- **一键生成 7 天计划**：早 / 午 / 晚（可选加餐），热量贴近目标、菜品不重样、避开忌口，每道菜带做法。
- **菜品手动替换 & 锁定**：任意一餐都能「🔄 换一道」从同类菜里替换；满意的菜可「🔒 锁定」，重新生成时保持不变。
- **方案保存与历史**：把满意的整周计划「💾 保存」并命名，之后随时在「🗂 历史方案」里载入或对比不同版本（存于本机）。
- **采购清单**：所有菜的食材按品类合并，并**按真实购买单位取整**（包 / 斤 / 板 / 盒 / 根 / 常备），可勾选、可一键复制。
- **批量预做建议**：标注哪些菜可一次做一周的量，建议周日集中备餐、分装冷藏/冷冻。
- **🏋️ 动作库（精选居家子集）**：内置约 110 个动作，覆盖胸/背/腿/腰腹/肩/手臂/有氧等 10 个部位，支持按部位、器械筛选与搜索，每个动作带中文说明、分步做法与动图（动图在线引用）。
- **🏋️ 每周训练计划**：选训练天数（3/4/5/6 天）与分化方式（全身 / 上下肢 / 推拉腿），一键排出一周训练；每条可「🔄 换」或「🔒 锁」，训练消耗自动计入「本周预测」。
- **本周预测（饮食 + 运动合一看板）**：纳入额外摄入（奶茶/咖啡/零食）、计划训练消耗与其他运动，实时算出净热量差与预计体重变化。
- **AI（可选）**：填入自带的模型 API Key（DeepSeek / OpenAI / 通义千问 / 自定义），用 AI 生成整周菜单并附搭配建议，App 内还有 AI 教练随时答疑；无 Key 时自动用本地算法兜底。

## 使用

直接用浏览器打开 `index.html` 即可。数据仅保存在你本机浏览器，无需联网、无需安装、不花钱。

> AI 功能需要你自己在「✨ AI 设置」中填入 API Key，Key 仅存于本地。
>
> 动作**文字数据**离线可用；动作**动图**来自在线仓库，需联网加载（离线时显示占位图标，不影响文字说明）。

## 数据与许可

- **菜谱库**：内置中式家常减脂菜谱，营养值为按常见做法估算，仅供参考，不作为医疗建议。
- **动作数据**：来自开源项目 [exercises-dataset](https://github.com/hasaneyldrm/exercises-dataset)。
